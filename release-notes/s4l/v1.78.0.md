## sim4life Application 8.2.0

## Added

- Added routing mechanisms for task manager and preferences windows
- Thumbnails for the advanced search video results & other improvements

## Changed

- Upgrade Python version to 3.11.9
- Updated react-router-dom version to latest version
- Moved bitrate indicator to bottom right and made it a link to relevant preferences
- Upgrade to `neuron 8.2.6`

## Fixed

- Array fields width now more consistent
- Inconsistent result in tools when searching in lower or upper case
- Fix issue when compiling Neuron mechanism files

## New services

- s4l-python runners (cpu/mpi and gpu) for running sim4life as computational job

---
## Platform Release Notes

#### Services
When a dynamic service is opened, it will now retrieve the inputs before displaying it's content.
Outputs are now deterministic, if the content of the files did not change the outputs will not be reuploaded to S3.

#### Improved data transfer feedback
When data is being downloaded to the service or uploaded from the service, an icon will be show it's state. Once it's completed, it will switch to a check icon and the data will be available.
